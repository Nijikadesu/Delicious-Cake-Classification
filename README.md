## Delicious_Cake_Classification

This is a Multi-target classifiction task using pretrained ResNet50

****
Environment required:
```angular2html
torch 2.2.2
torchvision 0.17.2
matplotlib 3.7.0
pandas 1.5.3
numpy 1.23.5
```
****
To fine-tune the ResNet50 model:
```angular2html
python main.py
```
To make inference:
```angular2html
python infer.py
```
****
Pretrained model are given as 'weights.pth'
